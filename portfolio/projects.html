DocuAgent

DocuAgent is an AI-based system designed to extract structured information such as text, tables, and charts from scanned and digital PDF documents. The system uses OCR techniques along with language models to convert unstructured documents into meaningful JSON data.

This project improved document processing efficiency and achieved high accuracy in character recognition and table extraction.

e-Samadhan

e-Samadhan is a web-based complaint management system developed for municipalities. It allows citizens to submit complaints online and track their status while enabling authorities to manage and resolve issues efficiently.

The system reduced complaint resolution time and improved transparency between citizens and government bodies.

AVL Tree Visualizer

This project is an educational web application that visually demonstrates insertion and deletion operations in an AVL tree. It helps students understand how self-balancing binary search trees maintain height balance.

The visual representation makes learning data structures more interactive and intuitive.
