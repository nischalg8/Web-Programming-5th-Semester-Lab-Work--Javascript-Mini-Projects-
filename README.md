# JavaScript Mini Projects (Lab Assignments)

This repository contains a collection of JavaScript mini projects given as lab assignments.  
Each project focuses on core JavaScript concepts such as DOM manipulation, events, arrays, objects, conditional logic, and dynamic UI updates.

## Projects Included

### Lab 1

#### Assignment 1: Blogging Website 
A multi-page blogging website using HTML and CSS.

#### Assignment 2: Personal Portfolio site
A resume or personal portfolio site.

### Lab 2 

#### Assignment 1: Smart Calculator with History
A basic calculator that performs arithmetic operations and stores the last 5 calculations in a dynamic history list. Includes invalid input handling and a clear history feature.

#### Assignment 2: Student Marks & Grade Analyzer
A system to store student details using objects, input marks for multiple subjects, calculate total/percentage/grade, and display results in a table. Pass/fail rows are highlighted using CSS applied via JavaScript.

#### Assignment 3: Dynamic To-Do List with Status Filter
A to-do application that supports adding, deleting, and marking tasks as completed. Includes filters for All, Completed, and Pending tasks, along with a completed task counter.

#### Assignment 4: Number Guessing Game with Score Tracking
A guessing game where the system generates a random number between 1 and 100. The user receives feedback (High/Low/Correct) and the score decreases with wrong attempts.

#### Assignment 5: Interactive Quiz Application
A quiz app where questions are stored as objects. One question is shown at a time, the user selects an answer, and the final score is displayed at the end.

#### Assignment 6: Digital Clock & Countdown Timer
A real-time digital clock and a countdown timer that accepts user input. The timer triggers an alert when the countdown reaches zero.

#### Assignment 7: Dynamic Table Generator
A program that generates a table dynamically based on user input (rows and columns). Includes buttons to add a row, delete a row, and highlight even rows.

#### Assignment 8: Simple E-Commerce Cart System
A cart system where products are stored as objects. Users can add/remove items, update quantity, and the total price updates dynamically.

#### Assignment 9: Form Validation System
A form validation project that validates name, email, and password fields. Displays error messages dynamically and prevents submission if validation fails.

#### Assignment 10: Event-Driven Color & Theme Manager
A theme manager that changes colors/themes using event-driven JavaScript logic.
